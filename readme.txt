Inventory management system. 

This is a simple inventory management system, that allows you to create and keep track of items in the store.

This project contains a users auth app, that allows users to be authenticated first before making use of the various 
functionalities. It also contains a store app that enables you to add an item to the store, issue out the item, receive
in more of the item in keep track of items in the store.

The project was built with Django (python), most attention was given more to the backend, and not the frontend. This is
to allow frontend developers to take their time to tweak the project the best way that they can.
